Windows下好看的TopBar yasb尝试
2025-04-27 08:00 folder 记录   label windows  

关注

在看B站视频时发现了这个漂亮的topbar，尝试装一下看看。

尝试

项目地址是 https://github.com/amnweb/yasb

需要先安装Nerd Fonts，官方推荐了一个字体包JetBrainsMono
下载后解压，然后选中所有字体文件，右键->安装。

接着安装程序，可以直接下载或者用winget等。

winget install --id AmN.yasb

启动就可以看到顶部topbar了

设置

运行后在系统托盘中会有一个图标

开机启动

右键 -> Enable Autostart

更换主题

右键 -> Get Themes，然后会出现主题列表预览并给你选择，直接Install就生效。

自定义配置

右键 -> Open Config，会打开配置文件目录，编辑config.yaml修改配置。
