macOS VNC连接提示No matching security types
2025-04-23 00:00 folder 问答   label macOS  

问题

macOS开启了屏幕共享，通过其他电脑用VNC连接时，提示
No matching security types when connect to MacOS VNC

回答

因为macOS的屏幕共享使用的是系统帐号验证，默认没有开启VNC密码验证。
解决方法是开启VNC密码验证，步骤如下：

  • 点开系统偏好设置，
  • 选择共享，
  • 然后在左侧选择屏幕共享，
  • 点击右侧的计算机设置...按钮，
  • 勾选VNC查看器可以使用密码进行访问，并设置一个密码。
