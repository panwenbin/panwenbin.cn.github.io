旧Mac安装新MacOS系统
2025-02-16 08:00 folder 记录   label macOS  

现状

有一台2012年中的MacBook Air (13-inch, Mid 2012)，标识为MacBookAir5,2，型号为A1466，内存4G，硬盘128G，官方支持的最新系统为macOS Catalina（10.15），但是从macOS Monterey（12）开始逐渐增加了随航等互联互通的功能，很多软件也需要系统版本12或13才能安装了，非常希望能升到较新的系统。

方案

通过检索，使用OpenCore Legacy Patcher工具可以安装官方不支持macOS系统。并且A1466的Intel Graphics HD 4000显卡刚好开始支持Metal，有显卡加速，流畅度较好。唯一限制较大的是只有4G内存。 系统则选择了Monterey（12），因为从Ventura（13）开始显卡加速变了，虽然OCLP v0.5.0开始有支持，但还是有一些问题的，详见Legacy Metal。

步骤

准备工作：一个U盘(不低于16G，如果15系统则需要大于16G)

  1. 下载OpenCore Legacy Patcher工具，官方网站 到Release页面下载OpenCore-Patcher-GUI.app.zip
  2. 打开OpenCore-Patcher-GUI.app，点击Create macOS Installer，然后选择Download macOS Installer，接着选择版本，我这边选择Monterey 12，下载完系统镜像会提示是否制作安装盘，此时把U盘插入，选择Flash Installer，根据容量选择对应的U盘，会自动格式化并写入系统镜像。写完后会提示制作引导，如果错过了，也可以从软件主界面选择Build and Install OpenCore，然后选择U盘，来制作引导。
  3. 完成后会提示重启，此时按住Option键，选择EFI boot，按Control+回车选择，进入后再选择Install macOS，之后就安装系统，安装过程中会重启几次，没问题的话最后会到登录界面或者初始化界面。
  4. 进入新系统，此时引导还在U盘上，打开OpenCore-Patcher-GUI，选择Build and Install OpenCore，此时选择系统硬盘，安装引导到硬盘上，然后重启，此时就可以直接从硬盘启动了。

总结

Monterey系统让我可以用Mac的键盘和触控板直接控制iPad2020，让两个老设备无缝配合，相当于有两个屏幕，iPad上可以运行网页或者iOS的App，Mac上可以运行工作软件。并且手不需要离开Macbook就可以操作。

其他

虽然Sequoia的iPhone镜像功能也很好用，但是需要Apple Silicon（M系）处理器或者带T2芯片才能使用。并且我尝试安装Sequoia最后并没有成功，卡在了一个升级画面就过不去了。
